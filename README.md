# 📋 Dashboard KPI — SAP Plant Maintenance

Dashboard en tiempo real que lee archivos Excel directamente desde OneDrive.

## Cómo actualizar los datos

1. Exporta los 4 reportes de SAP
2. Reemplaza los archivos en tu carpeta OneDrive:
   - `Avisos.xlsx` ← IW29
   - `Ordenes.xlsx` ← IW38
   - `IP16.xlsx` ← IP16
   - `IP24.xlsx` ← IP24
3. El dashboard se actualiza automáticamente (cada 5 min)

## Primera configuración

En el panel lateral de la app, pega los links de compartir de cada archivo:
1. Clic derecho en el archivo en OneDrive → Compartir
2. "Cualquier persona con el vínculo puede ver"
3. Copiar vínculo → pegar en la app
